GSPlay Documentation
====================

Real-time viewer for dynamic 4D Gaussian Splatting scenes.

**Version**: |version|

GSPlay is a high-performance viewer for PLY-based Gaussian Splatting sequences,
featuring real-time rendering, WebSocket streaming, and a web-based launcher for
managing multiple viewer instances.
